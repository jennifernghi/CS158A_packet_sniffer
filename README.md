# sniffer for CS158A

## Dependency

Require [yarn](https://yarnpkg.com/lang/en/docs/install/#mac-stable), Python 3.6+, [pipenv](https://github.com/pypa/pipenv#installation).

macOS:

```
brew install yarn pipenv python@3
```

## Setup 

```
cd $PROJECT_HOME
pipenv install
pipenv shell
pip install .
```

## Running

```
sniffer
```
